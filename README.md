# Weather Prediction using Random Forest

## Project Overview
This project predicts weather conditions using a Random Forest machine learning model.

## Dataset
The dataset contains historical weather information such as temperature, humidity, and wind conditions.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Model
Random Forest Classifier is used to train and predict weather conditions.

## How to Run
1. Download the notebook.
2. Open it in Jupyter Notebook.
3. Run all cells.

## Results
The model predicts weather conditions based on the input data.
